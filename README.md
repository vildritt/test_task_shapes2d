# Description

Simple application to collect and plot simple 2D shapes


# For developer

Coding style: "Qt Framefork-like"

## Dependencies

- CMake >= 3.17
- Qt5 >= 5.12 (without QML) or Qt6
    Tested versions: 5.12 (wo QML), 5.14.2, 6.2.1

## Build

Common cmake:

    $ cmake -B build
    $ cd build
    $ make
    [$ make test]
