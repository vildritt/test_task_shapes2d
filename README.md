# Description

Simple application to collect and plot simple 2D shapes


# For developer

Coding style: "Qt Framefork-like"

## Dependencies

- CMake >= 3.17
- Qt5 >= 5.12 or Qt6
    Tested versions: 5.12, 5.14.2, 6.2.1

## Build

    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    [$ make test]
